This was the problem
"Theorem succ_eq_add_one: For all natural numbers a, we have succ(a)=a+1."

And this was my solution
#+begin_src lean
rw[one_eq_succ_zero] // succ n = n + succ 0
rw[add_succ] // succ n = succ (n + 0)
rw[add_zero] // succ n = succ n
rfl
#+end_src

Pretty simple, makes sense.
